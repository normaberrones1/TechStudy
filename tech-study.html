What is the Java Virtual Machine (JVM) and its purpose?

The JVM is a virtual machine that executes Java bytecode. It allows Java programs to run on any device or operating system, providing cross-platform compatibility.

What is a variable in programming, and how do you declare a variable in Java?

A variable is a container for storing data values. In Java, a variable is declared by specifying its type followed by its name and optionally initializing it with a value.

EXAMPLE: int age = 25;

What are the primary data types in Java for representing whole numbers and decimal numbers?

For whole numbers: byte, short, int, long. For decimal numbers: float, double.

What are the primitive data types in Java, and what is the difference between primitive and reference data types?

The eight primitive data types in Java are: byte, short, int, long, float, double, char, and boolean. Primitive data types store simple values directly (e.g., int, boolean), whereas reference data types store references to objects (e.g., String, ArrayList).

What is numeric imprecision, and how can you avoid loss of precision when dividing integers?

Numeric imprecision occurs with floating-point types (float, double) due to their finite precision, which can lead to small errors in calculations. To avoid loss of precision when dividing integers, use floating-point literals or cast one operand to a double before performing the division.

What is widening conversion, and how does it differ from narrowing conversion?

Widening conversion is converting a smaller data type to a larger data type (e.g., int to long). Narrowing conversion is converting a larger data type to a smaller data type (e.g., long to int), and it requires explicit casting.
